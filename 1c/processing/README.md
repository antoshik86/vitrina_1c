# Создание внешней обработки "Выгрузка витрины на хостинг"

## Способ 1: Быстрый (Конфигуратор, 5 минут)

### Шаг 1. Создать внешнюю обработку
1. Открыть 1С в режиме **Конфигуратор**
2. `Файл → Новый → Внешняя обработка`
3. Имя: `ВыгрузкаВитриныНаХостинг`
4. Синоним: `Выгрузка витрины на хостинг`

### Шаг 2. Создать реквизиты
В ветке **Реквизиты** добавить:

| Имя | Тип | Длина |
|-----|-----|-------|
| Сервер | Строка | 100 |
| Порт | Число (5,0) | - |
| Пользователь | Строка | 50 |
| Пароль | Строка | 50 (Пароль = Да) |
| ПутьНаСервере | Строка | 100 |

### Шаг 3. Скопировать модуль объекта
Открыть вкладку **Модуль объекта** → скопировать весь код из:
→ `1c/processing/object_module.bsl`

### Шаг 4. Создать форму
1. ПКМ на ветке **Формы** → **Добавить**
2. Тип формы: **Форма объекта** (или обычная форма)

### Шаг 5. Разместить элементы на форме
Перетащить реквизиты на форму. Добавить кнопку "Выгрузить каталог на хостинг".

### Шаг 6. Скопировать модуль формы
Открыть вкладку **Модуль** формы → скопировать весь код из:
→ `1c/processing/form_module.bsl`

### Шаг 7. Сохранить
`Файл → Сохранить как...` → `vitrina_export.epf`

Готово. Можно открывать в режиме 1С:Предприятие.

## Использование

1. Открыть обработку в 1С:Предприятие (`Файл → Открыть`)
2. Заполнить настройки FTP (подставлены по умолчанию для `antonilyas.temp.swtest.ru`):
   - **Сервер:** `77.222.40.200`
   - **Порт:** `21`
   - **Пользователь:** `antonilyas`
   - **Пароль:** ваш пароль от хостинга
   - **Путь на сервере:** `/public_html`
3. Нажать **«Выгрузить каталог на хостинг»**

Обработка сама:
- Соберёт товары из справочника Номенклатура (УНФ)
- Сформирует `catalog.json` (для подгрузки через API)
- Сгенерирует HTML-карточки для каждого товара (для SEO)
- Загрузит всё по FTP на хостинг

После выгрузки — http://antonilyas.temp.swtest.ru/ обновится.
