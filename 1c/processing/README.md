# Создание внешней обработки "Выгрузка витрины на хостинг"

## Что делает обработка
- Собирает товары из справочника Номенклатура (УНФ)
- Формирует `catalog.json` (для сайта)
- Генерирует HTML-карточки для каждого товара
- Загружает всё по FTP на хостинг

## Инструкция

### 1. Создать внешнюю обработку
`Файл → Новый → Внешняя обработка`
Имя: `ВыгрузкаВитриныНаХостинг`

### 2. Создать реквизиты (ВНИМАНИЕ: обязательно!)
В ветке **Реквизиты** добавить 5 штук:

| Имя | Тип | Длина |
|-----|-----|-------|
| Сервер | Строка | 100 |
| Порт | Число (5,0) | - |
| Пользователь | Строка | 50 |
| Пароль | Строка | 50 (Пароль = Да) |
| ПутьНаСервере | Строка | 100 |

### 3. Скопировать модуль объекта
→ `1c/processing/object_module.bsl` — открыть вкладку **Модуль объекта**, вставить код

### 4. Создать форму
ПКМ на **Формы** → **Добавить** → Тип: **Форма объекта**

### 5. Разместить реквизиты и кнопку
**ВНИМАНИЕ:** В `form_module.bsl` форма строится программно (в `ПостроитьФорму()`).  
Поэтому элементы на форму **размещать не надо** — они создадутся сами.

### 6. Скопировать модуль формы
→ `1c/processing/form_module.bsl`

### 7. Сохранить
`Файл → Сохранить как...` → `vitrina_export.epf`
