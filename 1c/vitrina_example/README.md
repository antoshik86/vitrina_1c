# Внешняя обработка "Выгрузка витрины на хостинг" (через DFI)

Форма и реквизиты создаются **полностью кодом** через DFI. Ничего вручную!

## Содержимое папки

| Файл | Назначение |
|------|-----------|
| `../dfi/ModuleDFI.bsl` | Библиотека DFI — скопировать в новый общий модуль «ДФИ» |
| `object_module.bsl` | Модуль объекта обработки — скопировать как есть |
| `form_module.bsl` | Модуль формы — скопировать как есть |

## Инструкция

### Шаг 1. Создать общий модуль «ДФИ»
1. Конфигуратор → `Общие → Общие модули` → добавить
2. Имя: `ДФИ`
3. Галочки: **Клиент**, **Сервер**
4. Скопировать код из `1c/dfi/ModuleDFI.bsl`

### Шаг 2. Создать внешнюю обработку
`Файл → Новый → Внешняя обработка`
Имя: `ВыгрузкаВитриныНаХостинг_DFI`
**Никаких реквизитов не добавлять!**

### Шаг 3. Скопировать модули
- **Модуль объекта** → `1c/vitrina_example/object_module.bsl`
- **Модуль формы** → `1c/vitrina_example/form_module.bsl`

### Шаг 4. Сохранить и открыть
`Файл → Сохранить как...` → `vitrina_export.epf`
Затем `Файл → Открыть` — все поля уже на форме.

## Использование
1. Заполнить FTP (подставлены по умолчанию)
2. Нажать **«Выгрузить каталог на хостинг»**
