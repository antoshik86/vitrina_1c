# Пример: Витрина товаров (выгрузка из 1С на хостинг)

Данный пример демонстрирует использование библиотеки **DFI** для создания интерфейса внешней обработки, которая выгружает каталог товаров из 1С:УНФ на веб-сайт.

## Что делает обработка

1. Собирает товары из справочника **Номенклатура** (УНФ)
2. Формирует `catalog.json` для подгрузки через API
3. Генерирует HTML-карточки для каждого товара
4. Загружает всё по **FTP** на хостинг

## Как установить

1. Подключить общий модуль **ДФИ** в конфигурацию (скопировать код из `ModuleDFI.bsl`)
2. Создать внешнюю обработку:
   - `Файл → Новый → Внешняя обработка`
   - Имя: `ВыгрузкаВитриныНаХостинг_DFI`
   - Реквизиты: `Сервер (Строка)`, `Порт (Число)`, `Пользователь (Строка)`, `Пароль (Строка, Пароль=Да)`, `ПутьНаСервере (Строка)`
3. Скопировать код:
   - **Модуль объекта** → из `object_module.bsl`
   - **Модуль формы** → из `form_module.bsl` (форма создаётся автоматически через DFI)

## Как использовать

1. Открыть обработку в 1С:Предприятие
2. Настройки FTP уже заполнены по умолчанию (для `antonilyas.temp.swtest.ru`)
3. Нажать **«Выгрузить каталог на хостинг»**

## Зависимости

- Библиотека **DFI** (общий модуль `ДФИ`)
- 1С:Управление нашей фирмой (УНФ) или любая конфигурация со справочником Номенклатура
