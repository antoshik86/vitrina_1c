# Внешняя обработка "Выгрузка витрины на хостинг"

Форма строится **полностью кодом** — ничего ручного, никаких общих модулей.

## Файлы

| Файл | Куда копировать |
|------|----------------|
| `object_module.bsl` | **Модуль объекта** обработки |
| `form_module.bsl` | **Модуль формы** (весь DFI уже внутри) |
| `../dfi/ModuleDFI.bsl` | Справочно — оригинальная библиотека DFI |

В `form_module.bsl` встроены все нужные функции DFI в область `#Область DFI_ВспомогательныеФункции`. Никаких общих модулей, макетов, внешних компонент — просто копируете код.

## Инструкция

### 1. Создать внешнюю обработку
Конфигуратор → `Файл → Новый → Внешняя обработка`
- Имя: `ВыгрузкаВитриныНаХостинг`
- **Реквизиты НЕ добавлять**

### 2. Создать форму
ПКМ на ветке **Формы** → **Добавить** → Тип: **Форма объекта**
Форму закрыть, ничего не перетаскивать.

### 3. Скопировать код
- **Модуль объекта** → `object_module.bsl`
- **Модуль формы** → `form_module.bsl`

### 4. Сохранить и открыть
`Файл → Сохранить как...` → `vitrina_export.epf`
Открыть в 1С:Предприятие — все поля на форме уже есть.

## Как работает
- `ПриСозданииНаСервере` → создаёт 5 реквизитов через `ИзменитьРеквизиты()`
- Затем через встроенные DFI-функции строит группу, поля, кнопку
- Пользователь заполняет FTP и нажимает кнопку
- Обработка собирает товары, генерирует JSON + HTML, выгружает по FTP
