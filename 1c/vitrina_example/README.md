# Внешняя обработка "Выгрузка витрины на хостинг" (через DFI)

Форма строится **полностью кодом** через DFI. Ничего вручную — ни реквизитов, ни элементов формы.

## Требования

- 1С 8.3 (платформа 8.3.10+)
- Библиотека **ДФИ** (Динамическое Формирование Интерфейса) — https://github.com/KotovDima1C/DFI

## Установка

### 1. Подключить DFI
Скопировать `ModuleDFI.bsl` из репозитория DFI в конфигурацию как **общий модуль «ДФИ»** (клиент/сервер).

### 2. Создать внешнюю обработку
`Файл → Новый → Внешняя обработка`  
Имя: `ВыгрузкаВитриныНаХостинг_DFI`

**Никаких реквизитов не добавлять!**

### 3. Скопировать код
- **Модуль объекта** → `1c/vitrina_example/object_module.bsl`
- **Модуль формы** → `1c/vitrina_example/form_module.bsl`

### 4. Готово
Открыть `Файл → Открыть` → выбрать `vitrina_export.epf` — все поля уже на форме.
