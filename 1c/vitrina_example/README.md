# Пример: Витрина товаров (через DFI)

Полностью автоматическая внешняя обработка. Форма строится **кодом** через библиотеку **DFI**.

## Что нужно сделать в 1С

### 1. Подключить DFI
Скопировать `ModuleDFI.bsl` в новый **общий модуль** с именем `ДФИ`.

### 2. Создать внешнюю обработку
`Файл → Новый → Внешняя обработка`  
Имя: `ВыгрузкаВитриныНаХостинг_DFI`

**Реквизиты обработки — НЕ ДОБАВЛЯТЬ!**  
Форма сама создаст их динамически через `ИзменитьРеквизиты`.

### 3. Скопировать код
- **Модуль объекта** → из `object_module.bsl`
- **Модуль формы** → из `form_module.bsl`

### 4. Готово
Открыть обработку — все поля уже на форме.  
Нажать **«Выгрузить каталог на хостинг»**.

## Как работает

- `ПриСозданииНаСервере` → через `Форма.ИзменитьРеквизиты()` создаёт 5 реквизитов
- Затем DFI строит группу, поля ввода (с паролем), кнопку
- Пользователь заполняет FTP и нажимает кнопку
