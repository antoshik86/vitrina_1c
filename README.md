# Витрина 1С (УНФ) — статический интернет-магазин

**Сайт:** http://antonilyas.temp.swtest.ru/

Готовая вёрстка витрины товаров с корзиной и чатом для интеграции с **1С:Управление нашей фирмой (УНФ)**.

## Архитектура

```
[Хостинг (SpaceWeb / temp.swtest.ru)]  ←  [1С:УНФ (FTP + HTTP-сервисы)]
         ↓                                      ↓
   index.html, product.html                FTP: загрузка catalog.json
   css/style.js                            FTP: загрузка HTML-карточек
   js/api.js, cart.js, chat.js             POST /hs/vitrina/order
```

- **Фронтенд** — чистый HTML/CSS/JS. Хостится на SpaceWeb (бесплатно).
- **1С** сама генерирует JSON-каталог и HTML-карточки товаров и отправляет их по FTP на хостинг.
- **Заказы** — создаются в 1С через HTTP-сервисы как документы `ЗаказПокупателя`.
- **Чат** — AI-ассистент + передача менеджеру.

## Как всё настроено

### Сайт
- Хостинг: SpaceWeb (бесплатный)
- Адрес: http://antonilyas.temp.swtest.ru/
- FTP: `77.222.40.200`, порт `21`, логин `antonilyas`

### 1С → FTP-выгрузка на хостинг
Внешняя обработка `1c/vitrina_ftp.bsl`:
1. Загрузить в 1С (Файл → Открыть)
2. Нажать кнопку "ВыгрузитьНаХостинг"
3. 1С сформирует `catalog.json` и HTML-карточки товаров
4. Файлы улетят по FTP на хостинг

### 1С → HTTP-сервисы (заказы, чат)
Спецификация — `1c/HTTP_Services.md`.

Если 1С не поддерживает CORS — использовать PHP-прокси `api/proxy.php`.

## Как запустить

### 1. Обновить витрину из 1С
1. Открыть `1c/vitrina_ftp.bsl` в 1С
2. Проверить реквизиты FTP (логин/пароль)
3. Нажать "ВыгрузитьНаХостинг"
4. Зайти на http://antonilyas.temp.swtest.ru/ — товары обновлены

### 2. Настроить HTTP-сервисы (заказы)
В `1c/HTTP_Services.md` описаны все методы. Реализовать в конфигурации УНФ.

`js/api.js:3` — указать адрес сервера 1С.

### 3. AI-ассистент
В `POST /hs/vitrina/chat/ai` можно подключить:
- GigaChat (Сбер, бесплатно для РФ)
- OpenAI API
- Простой поиск по ключевым словам
